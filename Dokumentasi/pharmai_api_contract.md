# PharmAI: App-side Chat API Contract & Design

## 1. Overview
The PharmAI API supports persistent, multi-turn conversations and a one-off stateless chat mode for quick queries.

## 2. Technical Design

### Controller/Service Layer
- **[PharmaiChatController](file:///e:/Flutter/pharmvrpro/backend/app/Http/Controllers/Api/V1/Ai/PharmaiChatController.php#11-96)**: Handles request validation, authentication, and authorization via Policies.
- **[PharmAIService](file:///e:/Flutter/pharmvrpro/backend/app/Services/Ai/PharmAIService.php#13-159)**: Orchestrates AI logic, manages message persistence, and enforces domain restrictions.
- **[AiOrchestrator](file:///e:/Flutter/pharmvrpro/backend/app/Services/Ai/AiOrchestrator.php#5-41)**: Manages system prompts and context building for different AI modes.

### Conversation Flow
1. **Persistent**:
   - `POST /ai/conversations` -> Create session.
   - `POST /ai/conversations/{id}/messages` -> Send user input -> AI generates response -> Both stored in DB.
2. **Stateless**:
   - `POST /ai/chat` -> Send input + context -> AI generated response returned directly (not saved).

---

## 3. Endpoints

### 3.1. List Conversations
`GET /api/v1/ai/conversations`
- **Response**: `[ { "id": 1, "title": "GMP Basics", "last_message_at": "..." }, ... ]`

### 3.2. Create Conversation
`POST /api/v1/ai/conversations`
- **Body**: `{ "title": "Sterile Process Q&A" }`

### 3.3. Send Message (Persistent)
`POST /api/v1/ai/conversations/{id}/messages`
- **Body**: `{ "message": "Apa itu CPOB?" }`
- **Response**: AI message object with `role`, `content`, and `metadata`.

### 3.4. Stateless Chat
`POST /api/v1/ai/chat`
- **Body**: `{ "message": "Quick check for cleanroom class A", "context": { "current_module": "sterile" } }`
- **Response**: AI response object.

---

## 4. Message Roles
- [user](file:///e:/Flutter/pharmvrpro/backend/app/Models/UserAchievement.php#24-28): Messages sent by the learner.
- `assistant`: Responses generated by PharmAI.
- `system`: Internal context injection (e.g., training module state).
